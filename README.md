# 現場ツール

現場作業で使う小さな便利ツールです。インストールは不要です。

## 開き方

### Box Driveから開く場合

1. Box Driveのフォルダをエクスプローラーで開く
2. `index.htm`をダブルクリックする
3. EdgeまたはChromeで開く

BoxのWeb画面上でプレビューして開くのではなく、エクスプローラー上のBox Driveフォルダから開いてください。

### GitHubからダウンロードして使う場合

GitHubのページ:

https://github.com/kumasan007/tanabata

手順:

1. 上のページを開く
2. 緑色の`Code`ボタンをクリックする
3. `Download ZIP`をクリックする
4. ダウンロードされたZIPファイルを右クリックする
5. `すべて展開`をクリックする
6. 展開されたフォルダを開く
7. `index.htm`をダブルクリックする
8. EdgeまたはChromeで開く

注意:

- ZIPファイルのままではなく、必ず展開してから使ってください
- 展開後のフォルダ名を変えても使えます
- ただし、保存した内容を引き継ぎたい場合は、毎回同じ場所の`index.htm`を開く方が安全です

## 使えるツール

### コピペ

よく使う文字を登録して、カードをクリックするだけでコピーできます。

- 文字を入力して`追加`
- カードをクリックしてコピー
- 鉛筆ボタンで編集
- `×`ボタンで削除
- `≡`をドラッグして並び替え
- 登録内容はブラウザに保存

並び替えは2件以上ある時だけ使えます。

### スリーブ梁底計算

梁底からスリーブ芯までの距離を計算します。

入力する項目:

- 階高（FL）
- 梁スリーブ芯FL
- 梁成
- 下端フカシ
- スラブ天端レベル

できること:

- 入力するとすぐ結果が出る
- 結果をクリックしてコピー
- 入力欄右端の`×`でその欄を0にする
- `リセット`で全部0にする
- `保存`で現在の入力内容を履歴に残す
- 保存履歴から復元・削除できる

計算式:

```text
梁底FL = 階高（FL） - 梁成 - 下端フカシ + スラブ天端レベル
梁底からスリーブ芯までの距離 = 梁スリーブ芯FL - 梁底FL
```

## 保存について

入力内容や登録した文字は、使っているPCのブラウザ内に保存されます。

- 同じPC、同じブラウザで開けば前回の内容が残ります
- ブラウザの閲覧データを削除すると、保存内容が消える場合があります
- Box内のフォルダ名や場所を大きく変えると、保存内容が別扱いになる場合があります

## 注意

梁底計算は確認用の補助ツールです。最終判断は、設計図、構造図、社内基準、現場ルールに従って確認してください。
